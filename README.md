# Weather-Dashboard

# Description

Hi! In this weather dashboard you are able to search for the weather in a specific city. Once the search results appear you are also given a 5-day forecast.

Some of the weather conditions include:
- Temperature
- Humidity
- Wind Speed
- UV Index
